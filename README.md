# eflasher-updater
A console script that allows you to easily update eflasher



Usage
------------
```
git clone https://github.com/friendlyarm/eflasher-updater
cd eflasher-updater/
./update.sh
```
手动更新
------------
复制 bin/eflasher-gui 文件到SD卡root分区的/opt目录下，  
如果开发板已经通过SD卡启动到eflasher系统，则是覆盖开发板上的 /opt/ 下的同名文件。
